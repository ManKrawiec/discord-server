# discord-server

Prosta statyczna strona HTML + CSS dla serwera Discord "Bractwo Syntezy Świadomości".

## Struktura

- `index.html` - główna strona
- `css/style.css` - style
- `js/script.js` - prosty skrypt menu mobilnego
- `assets/` - grafiki
- `sections/` - pomocnicze pliki z treścią sekcji

## Publikacja na GitHub Pages

1. Wgraj pliki do repozytorium GitHub.
2. Wejdź w ustawienia repozytorium.
3. Otwórz sekcję `Pages`.
4. Wybierz branch `main` i folder root.
