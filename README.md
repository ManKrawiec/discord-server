# discord-server

Statyczna strona HTML/CSS/JS dla serwera Discord, przygotowana pod GitHub Pages w formie panelu z sidebar i dynamiczną zawartością.

## Struktura

- `index.html` - główny panel serwera
- `css/style.css` - pełny layout, responsywność i animacje
- `js/main.js` - przełączanie sekcji, hamburger menu i placeholder OAuth2
- `roles/roles.html` - osobna strona ról
- `channels/channels.html` - osobna strona kanałów
- `moderation/moderation.html` - osobna strona moderacji
- `members/members.html` - osobna strona członków
- `events/events.html` - osobna strona eventów
- `info/info.html` - osobna strona informacji i zasad
- `assets/icons/` - katalog na ikony
- `assets/images/` - katalog na grafiki

## Publikacja na GitHub Pages

1. Wypchnij repozytorium na GitHub.
2. Otwórz ustawienia repozytorium.
3. Wejdź w sekcję `Pages`.
4. Ustaw deploy z branch `main` i folderu root.
